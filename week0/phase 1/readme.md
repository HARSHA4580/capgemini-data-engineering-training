The main objective of this exercise is to gain hands-on experience with PySpark syntax and learn how data processing can be performed in a structured, step-by-step manner.

Dataset Overview

Customer Data
Includes basic customer information such as name, city, and age.

Orders Data
Contains details related to customer orders, including order amount and order date.

Key Learnings
Initializing and working with a Spark session
Creating DataFrames from sample data
Displaying and inspecting datasets
Applying filters to extract specific records
Selecting required columns from DataFrames
Performing grouping operations and counting records
